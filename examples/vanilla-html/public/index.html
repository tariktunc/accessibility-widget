Blakfy Accessibility Widget — Vanilla Demo

Bu sayfa hiçbir framework olmadan, sadece HTML + tek script tag ile widget'ı entegre eder.

Test Adımları

  1. Sol-altta FAB butonu görünür
  2. Tıklayın veya Alt + 0 ile panel açılır
  3. Tercih değiştirin, refresh sonrası korunmalı
  4. Sağ-altta "Powered by Blakfy Studio" rozeti

Programatik API

DevTools console:

window.BlakfyA11y.diagnostics()
